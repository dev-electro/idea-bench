# IdeaBench

This project provides 200+ validated startup ideas with detailed analysis. It was previously a paid product, now fully open-sourced.

## Features
- Browse startup ideas
- Preview a few sample ideas without logging in
- Stripe integration

## Tech stack
- Nextjs + React + Typescript
- Stripe for payments
- Tailwind / CSS modules for styling

## Usage
1. clone the repo
2. run `npm install`
3. add your stripe keys in `.env`
4. run `npm run dev` to start locally
